## 🛠️ Built With
- **HTML5**: Semantic structure.
- **CSS3**: Professional corporate design using modern Grid and Flexbox.
- **JavaScript**: Interactive UI components (Project Slider, Smooth Scroll).
- **Formspree**: Live contact form handling.
- **GitHub Pages**: Automated deployment and hosting.

## 📂 Project Structure
- `index.html`: Main page content and structure.
- `style.css`: Custom design system and responsive styles.
- `script.js`: Interactive logic and form handling.
- `assets/`: Images, screenshots, and professional documents.
